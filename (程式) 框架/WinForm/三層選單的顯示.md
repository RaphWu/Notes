# 三層選單的顯示

# 左側 TreeView 顯示兩層分類，右側 FlowLayoutPanel 顯示機台名稱勾選

你的思路非常清晰，這個設計 —— 左側用 `TreeView` 做兩層分類（**類別 → 型式**），右側用 `FlowLayoutPanel` 顯示勾選用的機台名稱（含 CheckBox）—— 是一個非常實用、空間彈性高、而且使用者友善的折衷方案，**沒有明顯疑慮，反而是非常推薦的實作方式之一。**

---

## ✅ 優點回顧：TreeView + FlowLayoutPanel 的組合

| 項目                        | 說明                                                                |
| ------------------------- | ----------------------------------------------------------------- |
| 📁 左側分類（TreeView）         | 清晰呈現「類別 → 型式」兩層分類結構                                               |
| ✅ 右側機台選擇（FlowLayoutPanel） | 可自由排列 CheckBox 控件，節省空間，視覺整齊                                       |
| 🧩 UI 適應性強                | FlowLayoutPanel 可隨視窗大小自動排版，適合小視窗                                  |
| 🧠 使用者直覺性強                | 點分類 → 顯示對應機台 → 勾選，流程簡單清楚                                          |
| 🎯 可維持原本架構                | 不用打掉重練，只需把兩層 `TabControl` 改成 `TreeView`，右側保留 FlowLayoutPanel 概念即可 |

---

## ❓ 你關心的幾個點

### 1. **TreeView 拉太長的問題解決了嗎？**

→ ✅ 有解決
因為你把機台列表移到右側 `FlowLayoutPanel`，TreeView 不需要再展開到第 3 層。它只顯示類別和型式，非常精簡。

### 2. **FlowLayoutPanel 放很多 Checkbox 會卡嗎？**

→ ✅ 不會卡，但請注意以下幾點最佳化：

* 使用 `SuspendLayout()` / `ResumeLayout()` 包裹更新流程，避免重繪多次：

```csharp
flowLayoutPanel1.SuspendLayout();
flowLayoutPanel1.Controls.Clear();

foreach (var machineName in machineList)
{
    var cb = new CheckBox
    {
        Text = machineName,
        AutoSize = true,
        Tag = machineName // 或可放ID
    };
    flowLayoutPanel1.Controls.Add(cb);
}

flowLayoutPanel1.ResumeLayout();
```

* 勾選後的資料可以用 `Checked` 屬性 + `Tag` 取回

### 3. **可否一次多選型式、顯示多個機台？**

→ 建議一次只顯示一個型式下的機台，避免 UI 混亂。TreeView 用單選模式即可（`AfterSelect` 事件處理）。

## 🧪 綁定 TreeView 範例（左側）

```csharp
treeView1.Nodes.Clear();

foreach (var category in result)
{
    var categoryNode = new TreeNode(category.Key);
    foreach (var type in category.Value)
    {
        var typeNode = new TreeNode(type.Key)
        {
            Tag = type.Value // 機台列表存在 Tag 中
        };
        categoryNode.Nodes.Add(typeNode);
    }
    treeView1.Nodes.Add(categoryNode);
}
```

## 📌 TreeView 點選事件 → 更新 FlowLayoutPanel

```csharp
private void treeView1_AfterSelect(object sender, TreeViewEventArgs e)
{
    if (e.Node.Level != 1) return; // 只處理型式層（第 2 層）

    var machines = e.Node.Tag as List<string>;
    if (machines == null) return;

    flowLayoutPanel1.SuspendLayout();
    flowLayoutPanel1.Controls.Clear();

    foreach (var machine in machines)
    {
        var cb = new CheckBox
        {
            Text = machine,
            AutoSize = true,
            Tag = machine
        };
        flowLayoutPanel1.Controls.Add(cb);
    }

    flowLayoutPanel1.ResumeLayout();
}
```

# 左側 TreeView + 右側 TabControl + FlowLayoutPanel，含延遲載入 CheckBox 清單

以下範例展示如何：

* 左側 `TreeView` 顯示第 1 層分類（Category）
* 右側 `TabControl` 顯示第 2 層分類（Type）TabPage，點擊時才動態產生該型式機台清單（CheckBox）
* 使用 `FlowLayoutPanel` 自動排列 CheckBox

---

## 假資料結構範例（模擬你的 Dictionary 結構）

```csharp
// 假資料，Dictionary<Category, Dictionary<Type, List<MachineName>>>
var data = new Dictionary<string, Dictionary<string, List<string>>>
{
    ["類別A"] = new Dictionary<string, List<string>>
    {
        ["型式A1"] = new List<string> { "機台A1-1", "機台A1-2", "機台A1-3" },
        ["型式A2"] = new List<string> { "機台A2-1", "機台A2-2" }
    },
    ["類別B"] = new Dictionary<string, List<string>>
    {
        ["型式B1"] = new List<string> { "機台B1-1", "機台B1-2" }
    }
};
```

---

## Form 設計要點

* 左側放一個 `TreeView`，名稱假設為 `treeViewCategories`
* 右側放一個 `TabControl`，名稱為 `tabControlTypes`
* 每個 `TabPage` 裡放一個 `FlowLayoutPanel`，用來放 CheckBox（動態建立）

---

## 程式碼示範

```csharp
public partial class MainForm : Form
{
    private Dictionary<string, Dictionary<string, List<string>>> _data;

    public MainForm()
    {
        InitializeComponent();
        InitializeData();
        InitTreeView();
        treeViewCategories.AfterSelect += TreeViewCategories_AfterSelect;
        tabControlTypes.SelectedIndexChanged += TabControlTypes_SelectedIndexChanged;
    }

    private void InitializeData()
    {
        _data = new Dictionary<string, Dictionary<string, List<string>>>
        {
            ["類別A"] = new Dictionary<string, List<string>>
            {
                ["型式A1"] = new List<string> { "機台A1-1", "機台A1-2", "機台A1-3" },
                ["型式A2"] = new List<string> { "機台A2-1", "機台A2-2" }
            },
            ["類別B"] = new Dictionary<string, List<string>>
            {
                ["型式B1"] = new List<string> { "機台B1-1", "機台B1-2" }
            }
        };
    }

    private void InitTreeView()
    {
        treeViewCategories.Nodes.Clear();
        foreach (var category in _data.Keys)
        {
            treeViewCategories.Nodes.Add(new TreeNode(category));
        }
    }

    // 當選擇左側類別，動態建立右側 TabPages（型式）
    private void TreeViewCategories_AfterSelect(object sender, TreeViewEventArgs e)
    {
        var category = e.Node.Text;
        if (!_data.ContainsKey(category))
            return;

        tabControlTypes.TabPages.Clear();

        foreach (var type in _data[category].Keys)
        {
            var tabPage = new TabPage(type);
            tabPage.Tag = type;
            // 加一個 FlowLayoutPanel 放 CheckBox
            var flow = new FlowLayoutPanel
            {
                Dock = DockStyle.Fill,
                AutoScroll = true
            };
            tabPage.Controls.Add(flow);
            tabControlTypes.TabPages.Add(tabPage);
        }
    }

    // 延遲載入 TabPage 的 CheckBox 清單
    private void TabControlTypes_SelectedIndexChanged(object sender, EventArgs e)
    {
        if (tabControlTypes.SelectedTab == null) return;
        var typeName = tabControlTypes.SelectedTab.Tag as string;
        if (typeName == null) return;

        var category = treeViewCategories.SelectedNode?.Text;
        if (category == null) return;

        var flow = tabControlTypes.SelectedTab.Controls[0] as FlowLayoutPanel;
        if (flow == null) return;

        // 若已載入（有控制項），則不重複載入
        if (flow.Controls.Count > 0) return;

        if (!_data.TryGetValue(category, out var typeDict)) return;
        if (!typeDict.TryGetValue(typeName, out var machines)) return;

        flow.SuspendLayout();
        foreach (var machine in machines)
        {
            var cb = new CheckBox
            {
                Text = machine,
                AutoSize = true,
                Tag = machine
            };
            flow.Controls.Add(cb);
        }
        flow.ResumeLayout();
    }
}
```

---

# 使用說明

1. 啟動程式後，左側 TreeView 顯示所有類別。
2. 點選某個類別時，右側 TabControl 會產生該類別所有型式的 Tab。
3. 切換 Tab 時，該 TabPage 內的 FlowLayoutPanel 才會產生對應機台的 CheckBox（延遲載入）。
4. 機台 CheckBox 可供使用者勾選。
