# 快捷鍵

{%preview https://learn.microsoft.com/zh-tw/visualstudio/ide/default-keyboard-shortcuts-in-visual-studio?view=vs-2022 %}

<p>您可以選擇適當的鍵盤快捷方式，在 Visual Studio 中存取各種 <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/reference/visual-studio-commands?view=vs-2022" data-linktype="relative-path">命令</a> 和視窗。 此頁面列出 <strong>[一般]</strong> 配置檔的預設命令快捷方式，您可能已在安裝 Visual Studio 時選擇該設定檔。 無論您選擇哪一個配置檔，都可以開啟 <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/identifying-and-customizing-keyboard-shortcuts-in-visual-studio?view=vs-2022#identify-a-keyboard-shortcut" data-linktype="relative-path">[<strong>選項</strong>] 對話框、展開 [<strong>Environment</strong>] 節點，然後選擇 [<strong>鍵盤]</strong>來識別命令的快捷方式</a>。 您也可以將不同的快捷方式指派給任何指定的命令，<a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/identifying-and-customizing-keyboard-shortcuts-in-visual-studio?view=vs-2022#customize-a-keyboard-shortcut" data-linktype="relative-path">自定義快捷方式</a>。</p>
<p>如需一般鍵盤快捷方式和其他生產力資訊的清單，請參閱：</p>
<ul>
<li>
              <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/productivity-shortcuts?view=vs-2022" data-linktype="relative-path">鍵盤秘訣</a></li>
<li>
              <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/productivity-features?view=vs-2022" data-linktype="relative-path">生產力秘訣</a></li>
</ul>
<p>如需 Visual Studio 中輔助功能的詳細資訊，請參閱 <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/reference/accessibility-tips-and-tricks?view=vs-2022" data-linktype="relative-path">輔助功能秘訣和訣竅</a> 和 <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/reference/how-to-use-the-keyboard-exclusively?view=vs-2022" data-linktype="relative-path">如何：以獨佔方式使用鍵盤</a>。</p>
<p>Visual Studio 中的鍵盤快捷方式可能會根據您的鍵盤配置和語言設定而有所不同。 您可以 <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/identifying-and-customizing-keyboard-shortcuts-in-visual-studio?view=vs-2022#identify-a-keyboard-shortcut" data-linktype="relative-path">識別命令的鍵盤快捷方式，</a> 或 <a href="https://learn.microsoft.com/zh-tw/visualstudio/ide/identifying-and-customizing-keyboard-shortcuts-in-visual-studio?view=vs-2022#customize-a-keyboard-shortcut" data-linktype="relative-path">自訂鍵盤快捷方式</a>。 如需詳細資訊，請參閱與此問題相關的 <a href="https://developercommunity.visualstudio.com/t/Issue-with-New-Ctrl---Comment-Shortc/10725157?q=keyboard+language&amp;ftype=idea&amp;stateGroup=active" data-linktype="external">開發人員社群票證</a>。</p>
<div class="heading-wrapper" data-heading-level="h2"><a class="anchor-link docon docon-link" href="#printable-shortcut-cheatsheet" aria-label="區段標題：可列印的快捷方式速查表"></a><h2 id="printable-shortcut-cheatsheet" class="heading-anchor">可列印的快捷方式速查表</h2></div>
