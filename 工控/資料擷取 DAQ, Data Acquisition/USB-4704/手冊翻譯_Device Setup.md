# Analog Input Configuration

Analog Input Configuration: Select "Analog Input" tab to configure analog input.

Item 1: Select ComboBox to set channel connection type, single-ended or differential for each channel.
設定每個頻道的連線類型，單端或差分。

Item 2: Select ComboBox to set value range type for each channel.
設定每個頻道的值範圍類型。

Item 3: Check the checkbox to configure all channels with the same connection type.
將所有通道配置為相同的連接類型。

Item 4: Check the checkbox to configure all channels with the same value range type.
將所有通道配置為相同的值範圍類型。

Item 5: Set section data count of Buffered AI.
設定緩衝式 AI 的分段資料計數。

Item 6: Select ComboBox to set convert clock source of Buffered AI.
設定緩衝式 AI 的轉換時脈來源。

Item 7: Set convert clock rate for single channel when clock source is internal.
設定時脈源為內部時單一通道的轉換時脈速率。

Item 8: Select ComboBox to set starting scanning channel of Buffered AI.
設定緩衝式 AI 的起始掃描頻道。

Item 9: Select ComboBox to set scanning channel count of Buffered AI.
設定緩衝式 AI 的掃描頻道數。

# Device Calibration

Note:
The USB-4704 has been calibrated at the factory for initial use. You are not required to calibrate the USB-4704 in normal conditions.
USB-4704 已在出廠時完成校準，可供初始使用。在正常情況下，您無需對 USB-4704 進行校準。

This section provides brief information on USB-4704 calibration. Regular calibration checks are important to maintain accuracy in data acquisition and control application. .
本節簡要介紹 USB-4704 的校準方法。定期進行校準檢查對於保持資料採集和控制應用的精確度至關重要。

To perform a satisfactory calibration, users need a 4-1/2 digit stable, low noise standard DC voltage source for the calibration process. It is important that the accuracy after calibration depends on the DC source's accuracy.
為了進行有效的校準，使用者需要一個 4-1/2 位元穩定、低雜訊的標準直流電壓源。校準後的精度取決於直流電壓源的精度，這一點非常重要。

USB-4704 Setting Calibration provides four functions :
- Manual A/D calibration
- Auto D/A calibration
- Manual D/A calibration

Please Click the "Calibration" tab in configuration dialog box and follow the Calibration Instructions to finish your calibration.

Item1 : Display Analog Input Calibration (Including Manual Calibration).
顯示類比輸入校準（包括手動校準）。

Item2 : Display Analog Output Calibration (Including Auto Calibration and Manual Calibration).
顯示類比輸出校準（包括自動校準和手動校準）。
