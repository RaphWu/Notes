# 更紗 Sarasa 字體

[GitHub](https://github.com/be5invis/sarasa-gothic)

- Style dimension
  - Latin/Greek/Cyrillic character set being - [Inter](https://github.com/rsms/inter)
    - Quotes (“”) are full width —— `Gothic`
    - Quotes (“”) are narrow —— `UI`
  - Latin/Greek/Cyrillic character set being - [Iosevka](https://github.com/be5invis/Iosevka)
    - Em dashes (——) are full width —— `Mono`
    - Em dashes (——) are half width —— `Term`
    - No ligature, Em dashes (——) are half width - —— `Fixed`
- Orthography dimension
  - `CL`: Classical orthography
  - `SC`, `TC`, `J`, `K`, `HC`: Regional orthography, - following [Source Han Sans](https://github.com/adobe-fonts/source-han-sans) notations.

# Sarasa 字體的樣式風格簡稱之參考

https://slashview.com/archive2022/20220428.html

1. `sarasa-mono-tc-semibold.ttf`
   半黑；建議套用在絕大部分的編輯器，例如|VSCode、VisualStudio、SSMS、Notepad...
2. `sarasa-mono-tc-bold.ttf`
   全黑；建議套用在對字體處理爛到爆的軟體，例如|LINE...

| 簡稱     | 樣式                                                                  |
| -------- | --------------------------------------------------------------------- |
| `sans`   | Sans-Serif；無襯線體                                                  |
| `slab`   | Slab-Serif；有襯線體，且襯線的樣式很像美國西部的風格                  |
| `gothic` | 全寬符號｜基於 Inter 字形                                             |
| `ui`     | 基於 Inter 字形｜半寬符號｜基於 Inter 字形                            |
| `mono`   | 全寬符號、等距等寬、有連字（例如大於等於合併成 ≧）｜基於 Iosevka 字形 |
| `term`   | 全寬符號、有連字｜基於 Iosevka 字形                                   |
| `fixed`  | 半寬符號、無連字｜基於 Iosevka 字形                                   |
| `CL`     | Classical Literature；傳統通用字形                                    |
| `SC`     | Simplified Chinese；簡體中文                                          |
| `TC`     | Traditional Chinese；繁體中文                                         |
| `HC`     | HongKong Chinese；香港繁體中文                                        |
| `J`      | Japanese；日文                                                        |
| `K`      | Korean；韓文                                                          |
