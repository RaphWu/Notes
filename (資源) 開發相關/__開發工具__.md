# Icons

## 圖庫

- [IconArchive](https://www.iconarchive.com/)
  擁有超過 75 萬個圖標，大多數都非常精美，而且很多圖標甚至可以免費用於商業用途。他們還提供圖標集，將風格相同的圖標分組，方便統一外觀。你只需在搜尋欄中輸入你想要的圖示類型即可。
- [DeviantArt](https://www.deviantart.com/)
  全球最大的線上藝術社區，擁有超過 4700 萬名註冊會員，其中包括藝術家和藝術愛好者。這裡匯集了來自世界各地藝術家創作的大量精美圖標，包括適用於 Windows 10 和 Windows 11 的（免費）桌面圖標和資料夾圖標。社群內容不斷更新，相信您一定能找到心儀之選。
- [Material Design icons from Google](https://fonts.google.com/icons)
  如果您喜歡 Android 及其圖標，Google 在其 Material Design 網站上提供了所有圖標。這些圖示簡潔明了，遵循 Google 的簡潔性和易用性設計理念。您可以下載任何您喜歡的 Google 圖標，但請注意，這些圖標是為 Android、iOS 和網頁設計的，因此僅提供 .png 和 .svg 格式。所以，如果您想在 Windows 11 或 Windows 10 中使用它們，則必須先將下載的圖示轉換為 .ico 格式。
- [FreeImages](https://www.freeimages.com/)
  您可以在這裡找到精美的桌面圖示以及其他各種圖示。您只需輸入描述所需 Windows 圖示的關鍵字，或瀏覽可用的圖示套件並下載您想要的圖示即可。
- [Dryicons](https://dryicons.com/)
  擁有大量獨特且專業設計的圖示。他們提供超過 6700 個圖標和向量圖形，並提供免費、普通或擴展許可，涵蓋各種風格、主題和顏色。
- [iconmonstr](https://iconmonstr.com/)
  其他網站略有不同。首先，他們提供大量圖標，但全部都是黑白的。您可以將它們用作桌面圖標、網站圖標，或用於演示文稿，以增強視覺效果。其次，他們提供的所有圖標格式均為 .svg、.eps、.psd 和 .png。這可能不太方便，因為這意味著您必須將它們轉換為 .ico 格式才能在 Windows 電腦上用作檔案圖示或資料夾圖示。
- [GraphicBurger](https://graphicburger.com/)
  如果您想下載高品質、像素級精準的圖標，GraphicBurger.com 是個不錯的選擇。該網站由羅馬尼亞設計師 Raul Taciu 經營，他提供的所有圖標和其他設計資源都可供任何人免費使用。不過，您需要注意的是，這些圖示的格式包括 .png、.svg、.eps 和 Sketch，如果您想在 Windows 10 或 Windows 11 中使用它們，則需要將它們轉換為 .ico 檔案格式。
- [Icons8](https://icons8.com/)
  這個網站提供了一個龐大的圖形庫，包括圖示、插圖、照片等等。它的免費圖示庫可謂包羅萬象。各種風格和類別應有盡有，您一定能找到適合您 Windows 電腦的自訂桌面圖示。不過，就像許多其他類似網站一樣，您需要知道免費圖示可以下載 .png、.svg 等檔案格式，但不能下載 .ico 檔案。這意味著您必須先將它們轉換為 .ico 格式，才能用作資料夾圖示、捷徑圖示或檔案圖示。
