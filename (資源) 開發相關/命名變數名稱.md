# 🧩 一、核心命名原則（簡單但很關鍵）

### 1️⃣ 固定順序（建議統一）

```
[功能][對象][方法][屬性]
```

例如：

- `ZeroDriftCompAvgWindow`
- `TempCompensationCoefficient`

---

### 2️⃣ 常用關鍵字統一

| 中文        | 建議用字                | 說明                   |
| ----------- | ----------------------- | ---------------------- |
| 補正 / 補償 | `Compensation` / `Comp` | 建議簡寫 `Comp`        |
| 零點        | `Zero`                  |                        |
| 飄移        | `Drift`                 |                        |
| 偏移        | `Offset`                |                        |
| 平均        | `Avg`                   | 不用 `Average`（太長） |
| 時間窗口    | `Window` ⭐             | 強烈推薦               |
| 時長        | `Duration`              |                        |
| 週期        | `Interval`              |                        |
| 係數        | `Coeff`                 |                        |
| 原始值      | `Raw`                   |                        |
| 濾波        | `Filter`                |                        |
| 溫度        | `Temp`                  |                        |

---

# 🧪 二、你的場景（零點飄移 + ADC平均）

### ✅ 建議一整組變數

```csharp
// ADC
double AdcRawValue;            // 原始ADC
double AdcAvgValue;            // 平均ADC

// 零點飄移補正
double ZeroDriftCompValue;     // 補正值
double ZeroDriftCompAvgWindow; // 平均時間窗口 ⭐

// 補正後
double CompensatedValue;       // 補正後結果
```

---

# 🌡️ 三、如果加上「溫度補償」（很常一起出現）

```csharp
double TempValue;              // 當前溫度
double TempCompCoeff;          // 溫度補償係數
double TempCompValue;          // 溫度補償量
```

👉 或更完整：

```csharp
double TempCompensationValue;
```

---

# 🌊 四、濾波 / 去抖動（工控很常用）

```csharp
double FilteredValue;
double FilterTimeConstant;
double MovingAvgWindow;
```

---

# ⚙️ 五、設定 vs 即時值（強烈建議區分）

### 設定值（UI / 參數）

```csharp
double ZeroDriftCompAvgWindowSetting;
```

### 運行中使用值

```csharp
double ZeroDriftCompAvgWindow;
```

👉 或更清楚分層：

```csharp
class Settings
{
    public double ZeroDriftCompAvgWindow;
}
```

---

# 🧱 六、進階（模組化命名，推薦）

如果專案變大，建議加「模組前綴」：

```csharp
double ZdCompAvgWindow;     // Zero Drift
double TempCompCoeff;
double AdcRawValue;
```

或用 class：

```csharp
class ZeroDriftCompensation
{
    public double AvgWindow;
    public double Value;
}
```
