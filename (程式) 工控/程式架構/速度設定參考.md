# 研華（Advantech）PCI-1220 軸控卡

在設定「低速／高速／加減速」時，其實是在定義馬達運動的三個核心參數：**起始速度、最大速度、以及速度變化曲線**。怎麼選，取決於你的機構、負載與應用目的。下面用實務方式幫你整理：

## 🔧 1️⃣ 低速（Start Speed / 初速）

👉 馬達開始動作時的速度

### 選擇原則

- **太低**：啟動平穩，但可能浪費時間
- **太高**：可能「起不動」或失步（步進馬達常見）

### 建議

- 步進馬達：設在「不會失步的最低穩定速度」
- 有負載時：稍微提高，但要測試是否抖動
- 常見做法：從低開始慢慢往上加，找到「剛好穩定啟動」的值

## 🚀 2️⃣ 高速（Max Speed / 最高速）

👉 運動過程中的最大速度

### 選擇原則

- 取決於：
  - 馬達能力（轉速上限）
  - 驅動器設定
  - 機構（皮帶、螺桿、慣量）

### 建議

- 不要直接設最大規格值
- 用「逐步提高測試法」：
  1. 從安全值開始
  2. 逐漸增加
  3. 觀察是否震動、失步或異音

👉 若出現問題，就略降 10～20%

## 🏁 3️⃣ 加減速（Acceleration / Deceleration）

👉 從低速到高速的變化斜率（速度爬升快慢）

### 選擇原則

- **太小（慢）**：
  - 動作平順，但整體時間變長

- **太大（快）**：
  - 容易震動、失步、甚至機構損壞

### 建議

- 有重載或高慣量：
  → 用「小加速度」
- 精密定位（如點膠、雷射）：
  → 偏小，確保穩定
- 一般搬運：
  → 可適度加大提高效率

## ⚖️ 實務調整流程（很重要）

建議用這個順序調整：

1. **先設低速** → 確保能穩定起動
2. **再找高速上限** → 找到不失步的最高值
3. **最後調加減速** → 在「不震動」與「效率」間取平衡

## 📊 簡單範例（概念）

假設單位是 pulse/sec：

- 低速：500
- 高速：5000
- 加速度：1000

👉 若出現：

- 起步抖動 → 降低低速或加速度
- 中途失步 → 降低高速或加速度
- 太慢 → 提高高速或加速度

## 🧠 補充關鍵觀念

- PCI-1220 是「脈波輸出卡」，實際速度還會受：
  - 馬達（步進 / 伺服）
  - 驅動器細分設定
  - 機構負載

- 所以**沒有一組通用數值，一定要現場調**

# 不同高速（PPU）下，一組合理的低速＋加減速搭配比例

工程上可用的搭配規則＋實際範例

## 🧠 核心觀念（先記這3點）

1. **低速 ≈ 高速的 5%～15%**
2. **加減速時間 ≈ 50～300 ms（常用範圍）**
3. **高速越高 → 加速度要越大（不然爬不上去）**

## 📊 常用搭配表（直接可用）

### 🚀 情境1：高速 PPU = 8000（偏高速運動）

👉 適合搬運、快速定位

- 低速：**400～800**
- 加速度：**2000～5000**
- 減速度：同加速度或略大（防止過衝）

#### 💡 建議組合

- 高速：8000
- 低速：600
- 加減速：3000

👉 特性：

- 起步穩
- 不會太拖
- 多數步進系統可用

### 🐢 情境2：高速 PPU = 400（低速精密）

👉 適合點膠、檢測、微動

- 低速：**50～100**
- 加速度：**100～300**

#### 💡 建議組合

- 高速：400
- 低速：80
- 加減速：150

👉 特性：

- 非常平順
- 幾乎無震動
- 但速度較慢

### ⚖️ 情境3：中速 PPU = 2000（通用）

👉 最常用區間

- 低速：**200～400**
- 加速度：**800～2000**

#### 💡 建議組合

- 高速：2000
- 低速：250
- 加減速：1200

## 🧮 更精準一點（工程公式思維）

其實你可以用這個簡化模型：

#### 1️⃣ 低速

```
低速 ≈ 高速 × 0.05 ~ 0.1
```

#### 2️⃣ 加速度（關鍵）

PCI-1220 的加速度單位通常是：
👉 pulse/sec²

估算方式：

```
加速度 ≈ 高速 / 加速時間
```

例如：

- 高速 = 8000
- 想 0.1 秒到達

👉 加速度 = 8000 / 0.1 = 80000

（但實務會設較小，避免太硬）

## ⚠️ 很多人踩的坑

#### ❌ 錯誤1：低速設太低

→ 馬達卡住、起步抖動

#### ❌ 錯誤2：加速度太大

→ 直接失步（最常見）

#### ❌ 錯誤3：高速很高但加速度太小

→ 根本跑不到高速（像慢慢爬）

## 🛠️ 實務調整「懶人流程」

直接照這個做最穩：

1. 固定高速（例如 8000）
2. 設低速 = 10%（800）
3. 從小加速度開始（500）
4. 慢慢往上加：
   - 沒震動 → 繼續加
   - 開始抖 → 往回降20%

## 🎯 快速總結（重點）

- 高速 8000 → 低速約 500～800，加減速 2000～4000
- 高速 2000 → 低速約 200～300，加減速 1000 左右
- 高速 400 → 低速約 50～100，加減速 100～300

👉 **一句話原則：**

> 「低速抓 5～10%，加速度抓『剛好不抖』的最大值」
